# EasyOrder4U
easyOrder4U Android app allows customers to order for sale items, food items, drinks &amp; beverages online from shops, business owners and restaurants near &amp; around them.

# Features
1. Geolocation
2. Notifications
3. Rich UI/UX
4. Firebase
5. Google Cloud
6. Paystack Payments Gateway
